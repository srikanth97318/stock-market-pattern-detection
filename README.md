# stock-market-pattern-detection
A Streamlit-based financial analytics dashboard that leverages linear algebra techniques such as normalization, correlation matrices, and eigen decomposition to identify hidden relationships and dominant trends in stock market data.
🧠 Key Features
📈 Real-time stock data using API
🔢 Matrix-based data representation
⚖️ Data normalization (Z-score standardization)
🔗 Stock correlation analysis using ( A \cdot A^T )
🧬 Eigenvalue & eigenvector computation
🔥 Dominant market trend detection
📊 Interactive visualizations using Streamlit
